# Sistema de Controle de Reservas de Hotel
Sistema desenvolvido em Java para gerenciamento de estadias, focado em aplicar conceitos fundamentais de Programação Orientada a Objetos (POO), como encapsulamento, manipulação de arrays e lógica de negócios.

###Funcionalidades:

Gestão de Reservas: Cadastro de hóspedes com escolha entre diferentes categorias de quartos (Normal, Premium e Luxo).

Cálculo Automático: Lógica integrada para definir o valor total da reserva baseado na diária da categoria e quantidade de dias.

Busca Inteligente: Localização de reservas específicas através do nome do hóspede.

Relatório Ordenado: Visualização de reservas organizada por tempo de permanência (ordem decrescente) utilizando Comparator.

Interface via Console: Menu interativo e intuitivo com validações de entrada.

###Conceitos Aplicados:

Encapsulamento: Getters e Setters para proteção de dados.

Arrays de Objetos: Gerenciamento de múltiplos registros em memória.

Expressões Lambda: Utilizadas para ordenação personalizada com Arrays.sort.

Tratamento de Fluxo: Uso de Switch-case, While e Do-while para controle de menu e validação de dados.
